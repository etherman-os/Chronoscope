## Known Limitations

1. **Landing Page Build Error**: The Next.js landing page has a pre-existing architectural issue where event handlers are passed to server components. This prevents `next build` from completing and is outside the scope of the security audit.
2. **Missing `package-lock.json` for Landing**: `services/landing/package-lock.json` is absent. It must be generated by running `npm install` in that directory on a macOS/Windows/Linux workstation and committing the result.
3. **Processor FFmpeg Dependency**: The processor crate requires system FFmpeg libraries (`libavcodec-dev`, `libavformat-dev`, `libavutil-dev`, `libswscale-dev`) which are not installed in this build environment.
4. **Go and Swift Compilers**: Not available in the current Linux environment. All Go and Swift changes were verified through careful manual syntax review.
